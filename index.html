cfDNA fragmentomics

MiNAE: Mutation-independent Nucleosome Architecture Evaluation

A liver cancer fragmentomics framework that learns from physical cfDNA fragment-size patterns instead of expensive tumor mutation sequencing.

  • Tests HCC against cirrhosis and hepatitis B controls
  • Includes ROC/PR curves, SHAP, ablation, and repeated validation

Target prioritization

PrISMA-RSC: Pancreatic immunotherapy target ranking

A specificity-aware consensus algorithm for prioritizing pancreatic cancer stromal targets while penalizing nonspecific candidates.

  • Integrates expression, spatial, structure, and control evidence
  • Stress-tests LRRC15 and comparator targets across scoring assumptions
Open project page
Epigenomic diagnostics

DNA methylation biomarkers for lung cancer detection

An AI-driven biomarker study using EGFR, PD-L1, SHOX2, RASSF1A, and PTGER4 methylation patterns across LUAD and LUSC cohorts.

  • 7,240 samples screened, 2,385 final processed samples
  • Gradient boosting model with 0.983 ROC-AUC and 98.2% recall
